# Amazon Vision Telegram Bot

Features:
- Identify product from image (GPT-Vision)
- Return names in EN / AR / FR
- Show 3 purchase buttons (local / .com / other)
- Track clicks per user
- Affiliate tag `chop07c-20` by default

## Env variables
Set these in Railway or .env:
- TELEGRAM_TOKEN
- OPENAI_API_KEY
- BASE_URL (public URL of your app, e.g. https://my-app.up.railway.app)
- AFFILIATE_TAG (optional, default chop07c-20)
- OPENAI_MODEL (optional)
